# Env-Template

### What is it?
This is a template of my default development environment.

---

### What does it do?
It saves the time I usually spend setting up my default development environment.

---

### What does it contain?
- React
- Vite 
- Tailwind CSS
- CLSX
- Tailwind-Merge
- React Router
- React Router Dom
- Boxicons
- CSS AOS Library
- Shadcn UI
- React Bits

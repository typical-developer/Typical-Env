# Env-Template

### What is it?
This is a template of my development environment.

---

### What does it do?
It helps save my time from manually installing new packages and dependencies each time I start a new project

---

### What does it contain?
- React
- Vite 
- Tailwind CSS
- CLSX
- Tailwind-Merge
- React Router
- React Router Dom
- Boxicons
- CSS AOS Library
- Shadcn UI
- React Bits
